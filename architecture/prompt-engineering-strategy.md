# Prompt Engineering Strategy

## Catalog

Prompt templates live under `prompts/` and are versioned with source control.

## Structure

Each prompt includes purpose, input, output format, constraints, review notice, assumptions, risks, and open questions.

## Hallucination Reduction

Prompts instruct the AI to avoid inventing facts, label recommendations, and identify missing stakeholder input.

## Evaluation Readiness

Future epics should add tests for prompt loading, template metadata, deterministic mock outputs, and artifact structure.
