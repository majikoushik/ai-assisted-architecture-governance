# Roadmap

## Completed

- Epic 0: Repository foundation.
- Epic 1: Architecture project workspace.
- Epic 2: Requirement submission.
- Epic 3: Prompt catalog and mock AI provider.
- Epic 4: Requirement analysis artifact.
- Epic 5: HLD generation.
- Epic 6: LLD and ADR generation.
- Epic 7: NFR, security, API, and risk reviews.
- Epic 8: Human review workflow and artifact versioning.
- Epic 9: Azure OpenAI provider readiness.
- Epic 10: Observability and production readiness.
- Epic 11: DevOps and Docker.
- Epic 12: Azure deployment blueprint.
- Epic 13: Portfolio polish, diagrams, docs, prompts, ADRs, sample outputs, and first-impression UI polish.

## Recommended Next Epics

1. Microsoft Entra ID authentication and role-based authorization.
2. Artifact version comparison and review diff view.
3. Prompt evaluation harness with golden outputs and safety checks.
4. Architecture review board workflow with approvals and decision logs.
5. Application Insights dashboard and alert rule templates.
6. PDF export and review pack generation.
7. Azure OpenAI managed identity hardening.
8. CI Bicep validation and optional container image publishing.

## Known Limits

- The app is a portfolio MVP, not a production governance system.
- Azure deployment templates require environment-specific review before deployment.
- AI outputs are drafts and require qualified human review.
- Authentication and authorization are documented future scope.
